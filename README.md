Ми успішно поєднали знання теорії графів із практичними навичками програмування на Python, розробивши функціональний скрипт для розв'язання задачі $3$-розфарбування. Головний внесок — це створення професійно організованої утиліти з візуалізацією та надійним інтерфейсом командного рядка (завдяки argparse), що значно підвищило її гнучкість та зручність використання.

Методи,які ми використовували:

  Для написання коду ми використали бектрекінг - алгоритм,який працює шляхом рекурсивного дослідження всіх можливих рішень проблеми. Він починає з вибору початкового рішення, а потім досліджує всі можливі розгалуження цього рішення. Якщо розгалження призводить до рішення, алгоритм повертає це рішення. Якщо розгалуження не призводить до рішення, алгоритм повертається до попередньої точки розгалуження та пробує інше шлях.Перевагами цього алгоритму є економія пам'яті та пришвидшене виконання,адже зайві варіанти не перевіряються.
