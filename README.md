***<h2>Бектрекінг</h2>***
Алгоритм бектрекінгу поступово перевіряє по черзі різні шляхи, і якщо шлях не підходить під умову завдання, то повертаються на крок назад і пробує інший, поки не знайде правильний. Працює методом рекурсивного дослідження всіх можливих рішень.Завдяки використанню бектрекінга можна пришвидшити виконання та зменшити обсяг використання пам'яті,адже він не перевірятиме зайві варівнти.

***<h2>Backtracking_without_colors</h2>***
Цей рекурсивний код, приймає файл без кольорів, а просто вершини записані через кому. За допомогою рекурсивного бектрекінгу ми знаходимо правильне розташування кольрів та записуємо в 
новий словник, який потім записуємо для більш простої візуалізації, в новий текстовий файл. Алгоритм також видає повідомлення, якщо розфарбувати граф 3 кольорами є нереальною задачею.
Данний алгоритм є основною задачею нашого проєкту.


***<h2>Backtrack_with_color</ha>***
Цей алгоритм вже не є обов'язковим для виконання, тому його реалізація є додатковою опцією для користувача. Принцип роботи полягає в тому, що користувач надає 2 файли, перший так само файл з вершинами, а наступний - це першопочаткове задання кольорів в графі. Оскільки задання кольорів може бути не правильне, то завдання алгоритму перевірити колір вершини з її сусідами та якщо проблем немає записати в новий файл, якщо ж виник конфлікт, то самостійно підібрати новий колір, який підходить під умови. Алгоритм також надає сповіщення користувачу, якщо граф неможливо розфарбувати в 3 кольори. Реалізація цього алгоритму, а також: його візуалізація та аргпарсинг є додатковим функціоналом.


***<h2>Візуалізація</h2>***
Для візуалізації розфарбованих графів ми використовуємо модулі matplotlib та networkx. Networkx створює обʼєкт граф, якому ми присвоюємо кольори. За допомогою matplotlib.pyplot ми створюємо зображення нашого графу. На зображенні відповідно названі вершини та показані їхні ребра. Надалі зображення у форматі .png завантажуються у файл, запропонований користувачем. 


***<h2>Користування програмою</h2>***
***<h3>Початковий граф - не розфарбований</h3>***
Для виконання програми введіть у термінал такий код:
``` python3 main.py --file testing.csv --img my_graph_result.png ```

Перший аргумент - файл з заданими ребрами графа.
Другий аргумент - файл, у який завантажується зображення (по замовчуванню 'graph.png').
  
***<h3>Початковий граф - вже розфарбований</h3>***
``` python3 main.py --file testing.csv --img my_graph_corrected.png --with_colors starting_wrong_colors.csv ```\
Перший аргумент - файл з заданими ребрами графа.
Другий аргумент - файл, у який завантажується зображення
Третій аргумент - файл, у якому задані початкові кольори вершин.
